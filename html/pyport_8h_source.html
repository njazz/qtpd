00416 /* Py_ARITHMETIC_RIGHT_SHIFT
00417  * C doesn't define whether a right-shift of a signed integer sign-extends
00418  * or zero-fills.  Here a macro to force sign extension:
00419  * Py_ARITHMETIC_RIGHT_SHIFT(TYPE, I, J)
00420  *    Return I >> J, forcing sign extension.  Arithmetically, return the
00421  *    floor of I/2**J.
00422  * Requirements:
00423  *    I should have signed integer type.  In the terminology of C99, this can
00424  *    be either one of the five standard signed integer types (signed char,
00425  *    short, int, long, long long) or an extended signed integer type.
00426  *    J is an integer >= 0 and strictly less than the number of bits in the

00433  */
